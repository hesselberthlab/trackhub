Description
-----------

Super track for DNA replication data for *S. cerevisiae*. 

Several tracks are available:

+ OriDB (www.oridb.org)
+ Replication timing by copy number (Yabuki et al.)
+ Replication timing by heavy / light (Raghuraman et al.)
